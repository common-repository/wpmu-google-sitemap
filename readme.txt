== Description ==

Adds functionallity for Wordpress 3 to create a Google Sitemap XML for all content on the the current installation. Please note that this will add links to all public pages, posts and blogs on the domain.

== Installation ==

1. Unzip the file and upload the folder to `/wp-content/plugins/`
1. Activate the plugin for the MAIN BLOG ONLY
1. Visit the `/wp-admin/admin.php?page=wpmugs-settings` and save the changes
1. Add the sitemap URL to Google Webmaster Tools or whatever service you like to use
